### 📈 Key Findings (4 Major)

**Finding #1**: Guardrails don't hurt accuracy
- F1 improves from 0.393 → 0.396
- Balanced accuracy improves from 61.6% → 62.0%
- Only 2.8% recall trade-off for 66% compliance improvement

**Finding #2**: Regulatory model immune to adversarial text
- Baseline fooled on 7 risky cases (0.13% attack success)
- Regulatory fooled on 0 risky cases (0.0% attack success)
- Numeric + flags override text manipulation

**Finding #3**: Regulatory flags are predictive
- No violations: 19.6% default rate
- Critical violations: 42.9% default rate
- 2.2× risk correlation validates regulatory constraints

**Finding #4**: Operating point optimization matters
- τ = 0.25: Conservative (93.7% recall, 1.4% violations)
- τ = 0.35: Balanced (82.0% recall, 1.4% violations) ← RECOMMENDED
- τ = 0.50: Risky (59.5% recall, 10.1% violations)
